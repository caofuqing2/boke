## for循环
解析：十行十列
1. 先循环十行
2. 再循环十列
3. i循环执行一次，j循环执行十次。
```
for(var i = 0; i < 10; i++){
console.log('这是第'+i+'行');
// i循序执行一次 j循环会执行10次
for(var j = 0; j < 10; j++) {
console.log('这是第'+j+'列');
}
}
```

计算top值：当前是第n行，n*行高

计算left值：当前是滴n行，n*列高
